# Masking Slider

オブジェクトからマスクを用いて切り出し、円形スライダー状に回転させるスクリプトです。

現在は **version 0.11** が最新です。

以下のスクリプトが備わっています。

## MaskingSlider

以下はパラメーターの説明です。

- **トラックバー/チェックボックス**
  - マスク数
    - オブジェクトからマスクへ切り出す数を指定します。
  - 幅
    - マスクのライン幅を指定します。
  - マスクtype
    - マスクの図形の形を指定します。
