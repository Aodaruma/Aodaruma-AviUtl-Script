# 平行四辺形

アンカーを用いて、図形を平行四辺形に変形させるスクリプトです。

現在は **version 1.01** が最新です。

以下のスクリプトが備わっています。

## 平行四辺形

以下はパラメーターの説明です。

- **トラックバー/チェックボックス**
  - affect%
    - 元の図形と平行四辺形の変形度を設定します。
  
- **ダイアログ**
  - pos
    - アンカーポイントの位置を保存する配列です。
