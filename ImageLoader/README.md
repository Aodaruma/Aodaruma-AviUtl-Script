# ImageLoader

最大画像サイズより大きい画像を読み込むことができるプラグインです。

現在は **version 0.1** が最新です。

以下のスクリプトが備わっています。

## ImageLoader

以下はパラメーターの説明です。

- **トラックバー/チェックボックス**
  - ｴｯｼﾞﾀｲﾌﾟ
    - 画像のエッジを補間する種類を選択します。
  - 補間ﾀｲﾌﾟ
    - 画像の拡大・縮小に用いる補間アルゴリズムの種類を選択します。
  - ｷｬｯｼｭ保存
    - 一度読み込んだら、キャッシュとして保存しておくか選択します。
  
- **ダイアログ**
  - ファイルダイアログ
    - 読み込む画像のパスを設定してください。
